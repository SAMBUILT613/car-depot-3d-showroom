# Car Depot 3D Showroom

A premium, interactive 3D landing page for **Car Depot | Ultimate Autos NG**, a used car dealership based in Lagos, Nigeria. This project serves as a high-end portfolio piece demonstrating advanced web technologies like Three.js and GSAP without the need for complex build pipelines.

## ✨ Features
- **3D Interactive Showroom:** Explore a high-quality car model in a real-time 3D environment using Three.js.
- **Scroll-Driven Animations:** Smooth section transitions and reveals powered by GSAP ScrollTrigger.
- **Mobile Optimized:** Responsive design that performs well on mobile devices.
- **Premium Aesthetic:** A dark, modern "automotive brand" look and feel.

## 🛠️ Tech Stack
- **Three.js v0.160.0:** 3D rendering engine.
- **GSAP 3.12.5:** Animation library for scroll effects and UI transitions.
- **Vanilla JS (ES Modules):** Clean, modern JavaScript without bundlers.
- **CSS3:** Custom styling with modern features like CSS Variables and Flexbox/Grid.
- **Netlify:** Fast, reliable static site hosting.

## 📦 Local Development
Since this project uses ES Modules and Three.js loaders, it requires a local server to run correctly due to CORS restrictions.

1. **Using Node.js:**
   ```bash
   npx serve
   ```
2. **Using Python:**
   ```bash
   python3 -m http.server
   ```
3. Open `http://localhost:3000` (or the port provided) in your browser.
